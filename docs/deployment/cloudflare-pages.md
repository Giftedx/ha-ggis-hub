Notes:

- `wasm-unsafe-eval` required for WebAssembly compilation under CSP.
- `style-src 'unsafe-inline'` retained for style attributes generated by Vite; tighten if practical.

- HSTS preload enabled; ensure all subdomains are HTTPS before pointing additional subdomains at this project.

## Shipped `public/_redirects`

WHS is a separate deployment (not mounted under the hub build), so only the SPA fallback is present:

```text

```

If WHS is ever served from a sub-path of this Pages project, add its specific route **before** the wildcard:

```text
/wild-haggis-survivors/*  /wild-haggis-survivors/index.html  200
/*  /index.html  200
```

## Cache policy

- `index.html`: revalidate every request.
- hashed assets: one year immutable.
- manifest: short cache.
- un-hashed files: no long immutable cache.

## Source maps

Production default: no public source maps.

Preview default: source maps allowed.

If hidden maps are generated for error tracking, they must be uploaded privately and removed from public `dist` before deployment.
